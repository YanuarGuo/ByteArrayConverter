# Byte Array Converter in C#

A simple C# application to convert text and images to byte arrays and vice versa.

## Features
- Convert text to a byte array and back
- Convert images to a byte array and save them back as images
- Supports common image formats (JPEG, PNG, BMP, GIF)
- Command-line interface for easy usage

## Prerequisites
- .NET 8.0 or later

## Installation
1. Clone this repository:
   ```sh
   git clone https://github.com/YanuarGuo/ByteArrayConverter.git
   ```
2. Open the project in Visual Studio or any C# IDE of your choice.

## Usage
### Convert Text to Byte Array and Back
This will output the byte array representation of the text and convert it back to the original text.

### Convert Image to Byte Array and Save Back
This will read the image as a byte array and save it back as an image.
